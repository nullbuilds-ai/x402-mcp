Tool reference

All 7 tools exposed by x402-mcp. Discovery and estimation require no credentials. Payment tools require a CDP wallet configured via x402_configure.

x402_discover
Browse the API catalog. Filter by keyword or network to find x402-enabled APIs. Returns live endpoints with pricing and network details.
free
Parameters
Name Type Description
keyword string optional Search term to filter APIs by name or description. E.g. "weather", "FDA".
network string optional Filter by blockchain network. E.g. "base", "base-sepolia".
Returns List of APIs with endpoint URLs, cost per call, and network.
x402_estimate
Check the exact cost of an API call before committing any payment. Probes the endpoint live and returns the payment requirements.
free
Parameters
Name Type Description
url string required Full URL of the x402-enabled endpoint to probe.
Returns Price in USDC, payment token contract address, and network.
x402_pay
Make a paid API call. Handles the full x402 payment flow: wallet signing, payment header encoding, and request retry. Returns the API response and a payment receipt.
CDP wallet required
Parameters
Name Type Description
url string required Full URL of the x402-enabled endpoint to call.
method string optional HTTP method. Defaults to "GET".
body string optional Request body for POST/PUT requests. Pass as a JSON string.
Returns API response body plus a receipt with amount paid, transaction hash, and timestamp.
x402_balance
Check the USDC balance of the configured CDP wallet. Reads from the wallet set up via x402_configure.
CDP wallet required
Parameters
none
Returns USDC balance of the configured wallet.
x402_configure
Configure the MCP server with a CDP wallet and network. Required before using x402_pay or x402_balance. Signing happens in Coinbase's TEE — your private key never touches your machine.
CDP wallet required
Parameters
Name Type Description
wallet_seed string optional CDP wallet seed phrase or private key. Stored in memory only for the session.
network string optional Target network for payments. E.g. "base", "base-sepolia".
Returns Configuration status and active wallet address.
x402_catalog_stats
Get current statistics for the API catalog. Useful for understanding catalog freshness and scale before running discovery queries.
free
Parameters
none
Returns Total live APIs, total endpoints tested, and last updated timestamp.
x402_health
Health check for the MCP server. Returns server status and current version. Useful for verifying the server is running correctly after installation.
free
Parameters
none
Returns Server status and version string.
